# IdeaValidator — Site Vision & Roadmap

## 1. Overview
**IdeaValidator** is a SaaS platform that uses AI to validate business ideas with deep market analysis, competitor intelligence, and exportable reports.

**Stitch Project ID:** `3087119646706204728`
**Design System:** Luminal Observer (Dark Mode — `.stitch/DESIGN.md`)
**Device:** DESKTOP

---

## 2. Tech Stack
- React + Vite (frontend)
- Node.js/Express (backend, port 3001)
- Framer Motion (animations)
- React Router DOM (routing)

---

## 3. App Sections (Routes)
| Route | Component | Description |
|-------|-----------|-------------|
| `/` | Landing | Hero landing page with idea input |
| `/report` | ReportDashboard | Main dashboard with overview scores |
| `/breakdown` | ScoreBreakdown | Detailed analysis by dimension |
| `/radar` | CompetitorRadar | Competitor intelligence matrix |
| `/export` | Export | Export hub for PDF/JSON/CSV |
